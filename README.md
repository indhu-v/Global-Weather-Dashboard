# Global-Weather-Dashboard
A fully interactive Excel-based weather analytics dashboard built using Pivot Tables, Pivot Charts, and Slicers. This dashboard visualizes global weather patterns, including temperature, humidity, sunrise/sunset times, UV index, and weather conditions for multiple countries.
